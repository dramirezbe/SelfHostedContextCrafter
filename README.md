# 🛠️ SelfHostedContextCrafter (shcc)

**The local-first, open-source agent framework that engineers perfect project context and audits complex codebases.**

SelfHostedContextCrafter does not write code. It acts as an automated architectural team that runs completely on your local hardware (via Ollama/Llama.cpp). By orchestrating RAG-driven expert agents, it digests dense hardware datasheets, software requirements, and physical specifications to output a hyper-focused project scaffold, directory tree, and localized context briefs. 

Once you begin development, it serves as your local inspector—scoring your code and suggesting refactors against your original hardware and software constraints.
